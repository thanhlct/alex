Instructions ▾

You are given a task which will require you to call an automated phone service and request information about a public transport connection from one place to another, about the number of transfers, next connection, time of departure, weather across the US, what is the time, etc.

You can say "Help" for hints on how to use the service. If the system seems confused, you can say "Reset" or "Start over" to start over. The service can pick up what you are saying and ask you questions as the dialogue goes along.

  • Make sure you are in a quiet place and turn off disturbing elements such as TV for the system may pick up sources of noise and understand you poorly.
  • After you are connected, please try to talk naturally about the topic presented in the green box at the bottom.
  • Talk in whole sentences, don't say just key words.
  • The call is finished by saying "Thank you, Goodbye" to the system.
  • At the end of the call, the system will give you a four-digit code which you have to enter into a field down below, once the call is terminated. This is used to verify the authenticity of Job submissions.
  • You will not be given any code if you do not exchange at least three sentences with the system, so do not rush the dialogue towards the end too quickly. Just chat a little.
  • If a correct code is entered, you can fill in a short feedback form.
  • Your code will only be valid for 15 minutes, so be sure to fill in the form right after you finish the call.
  • You are more than welcome to enter general comments and observations.

The feedback is very important as only the calls with the correct code and completed feedback will be accepted.
